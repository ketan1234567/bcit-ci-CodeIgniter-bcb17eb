1) Return Single Object (Your current code)

return $query->row();

What it means (simple English)

Returns only one record

Result is an object

Use when:

Username is unique

Login, profile, single record

Access example

echo $user->username;
echo $user->role;
___________________________________________________________________________________________________________________________________

2) Return Single Row as Array

return $query->row_array();

Meaning

One record

Returned as an array

echo $user['username'];

___________________________________________________________________________________________________________________________________

3) Return Multiple Records as Objects

return $query->result();

Meaning

Returns many rows

Each row is an object

___________________________________________________________________________________________________________________________________

4) Return Multiple Records as Array

return $query->result_array();

Meaning

Many rows

Each row is an array

foreach ($users as $u) {
    echo $u['username'];
}
___________________________________________________________________________________________________________________________________

5) Return Boolean (true / false)

return $query->num_rows() > 0;


Meaning

Just checks if record exists

Used for:

Username exists check

Validation

___________________________________________________________________________________________________________________________________

6) Return Count (Number)

return $query->num_rows();

Meaning

Returns how many rows found


___________________________________________________________________________________________________________________________________

7) Return Custom Data

return [
    'user' => $query->row(),
    'count' => $query->num_rows()
];

___________________________________________________________________________________________________________________________________

| Use case            | Best return      |
| ------------------- | ---------------- |
| Login / single user | `row()`          |
| User list           | `result()`       |
| API JSON            | `result_array()` |
| Exists check        | `num_rows()`     |
| Dashboard stats     | custom           |
